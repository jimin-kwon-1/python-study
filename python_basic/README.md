# Python Study Notes

This repository contains my Python practice codes organized by topic.
It is a personal study record covering Python basics, data structures,
functions, file handling, exceptions, and simple generators.

## Structure
- `01_basics_strings.py`  
  String operations, formatting, slicing, and basic operators

- `02_control_flow.py`  
  if / for / while statements and iteration patterns

- `03_collections.py`  
  List and dictionary usage, nested structures, and counter patterns

- `04_functions.py`  
  Functions, default arguments, *args, recursion, lambda, map/filter

- `05_recursion_fibonacci.py`  
  Recursive Fibonacci with memoization

- `06_files.py`  
  File read/write, random data generation, and simple BMI processing

- `07_exceptions_and_generator.py`  
  Input validation, try-except handling, and generator basics

## Notes
- Some codes are written for learning purposes and may be verbose.
- Error cases and edge cases are intentionally included during practice.

## Environment
- Python 3.x
- Written and tested in VS Code
